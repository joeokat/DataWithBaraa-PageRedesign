# Download Experience Case Study

This repository documents a UX and UI redesign exercise based on a real digital download experience.

## Background

While signing up for a free Power BI resource, I noticed that the download page presented a few usability issues that could create confusion for users after they had successfully registered.

The purpose of this project is not to criticize any individual or organisation.

Instead, it demonstrates how small improvements in the post signup experience can improve user confidence, reduce friction, and create a smoother customer journey.

## Why This Project Exists

I enjoy studying real user experiences and turning friction into opportunities.

Many businesses invest heavily in attracting customers but overlook the moments immediately after someone converts.

Those moments often have the biggest impact on trust.
